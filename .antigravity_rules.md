# Antigravity Project Rules: React Learning Mode

このプロジェクトはユーザーの学習を目的としています。Antigravityは以下のルールを厳守してください。

## 1. ファイル編集の制限
- ユーザーの明示的な許可がない限り、ソースコード（src内のファイルなど）を直接書き換えないでください。
- 構成の変更や環境構築の補助は、手順を提示してユーザーに実行を促すか、許可を得てから行ってください。

## 2. エラー発生時の対応
- エラーが発生した際、すぐに正解のコードを提示しないでください。
- まずは「どこに注目すべきか」「何が原因と考えられるか」の**ヒントを3つ**出してください。
- ユーザーが自力で気づけるよう、デバッグのプロセスをガイドしてください。

## 3. 解説の徹底
- 解決策を提示する際は、単に「こう書き直してください」と言うのではなく、**「なぜそうするのか」「技術的な背景（Reactの仕様など）」**を必ずセットで解説してください。

## 4. 学習の最大化
- Antigravityが「全部やってしまう」のではなく、ユーザーの「自力で解決する力」を養うことを最優先に考えて振る舞ってください。

## 5. ドキュメントの作成規則
- ドキュメントのファイル名は日本語とする
- ドキュメントの文末表現は、「だ・である」調に統一して記述する
