# ReactStudy

Reactの学習過程を記録し、技術的な理解を深めるためのプロジェクトである。

## プロジェクトの目的
Vanilla JavaScriptの基礎からReactの応用概念まで、段階的に学習を進めることを目的とする。
学習した内容は `docs/` ディレクトリ内に詳細なドキュメントとして整理されている。

## プロジェクト構造
- `docs/`: 学習した技術コンセプトのまとめ（だ・である調）
- `vanilla_js/`: JavaScript基礎の練習環境
- `react/`: Reactコンポーネントやアプリケーションの学習環境

## 開発ルール (.antigravity_rules.md)
本プロジェクトでは、AIアシスタント（Antigravity）との共同作業において以下のルールを設けている。

1. **学習優先の姿勢**: AIが直接コードを書き換えるのではなく、ヒントやデバッグのガイドを行うことで、自力での解決能力を養う。
2. **エラー時のヒント**: エラー発生時は、解決策を即答せず、3つのヒントを提示する。
3. **技術的背景の解説**: 解決策の提示には、必ず「なぜそうなるのか」という背景解説をセットにする。
4. **ドキュメント規定**: 
   - ファイル名は日本語を使用する。
   - 文末表現は「だ・である」調に統一する。
