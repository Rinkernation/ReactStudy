# JS基礎: 三項演算子

`if / else` 文による条件分岐を、たった1行の式で完結に記述するための構文が **三項演算子** である。

## 1. 基本的な構文

三項演算子は以下の形で記述する。

```javascript
条件式 ? 条件がtrueの時の値 : 条件がfalseの時の値
```

クエスチョンマーク（`?`）の左側に条件を書き、コロン（`:`）で区切って「真の場合」と「偽の場合」の処理を記述する。

## 2. 具体的なコード例

### 変数への代入
条件によって変数に代入する値を変えたい場合、`if / else` を使うとコードが長くなるが、三項演算子なら1行でスッキリと書ける。

```javascript
// 例1: 数値の判定
const val1 = 1 > 0 ? "trueです" : "falseです";
console.log(val1); // "trueです"

// 例2: 型の判定とフォーマット
// 変数 num が数値型であればカンマ区切りにし、そうでなければメッセージを返す
const num = 1300;
const formattedNum = typeof num === 'number' ? num.toLocaleString() : '数値を入力してください';
console.log(formattedNum); // "1,300"
```

### 関数の戻り値としての利用
関数の `return` においても、三項演算子を使うことで簡潔に結果を返すことができる。

```javascript
const checkSum = (num1, num2) => {
  return num1 + num2 > 100 ? "100を超えています!!" : "許容範囲内です";
};

console.log(checkSum(50, 40)); // "許容範囲内です"
```

## 3. Reactでの活用シーン（条件付きレンダリング）

React（JSX）の中では、直接 `if` 文を記述することができない仕様になっている。
そのため、「フラグが true なら A のコンポーネントを表示し、false なら B のコンポーネントを表示する」といった**条件付きレンダリング**を行う際、この三項演算子が必須のテクニックとなる。

```javascript
// Reactでの使用例のイメージ
const App = ({ isLogin }) => {
  return (
    <div>
      {/* isLoginがtrueならマイページを、falseならログイン画面を表示 */}
      {isLogin ? <MyPage /> : <LoginScreen />}
    </div>
  );
};
```
