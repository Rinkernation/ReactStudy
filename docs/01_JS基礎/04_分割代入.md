# JS基礎: 分割代入

React開発において、オブジェクトや配列から必要なデータをスマートに取り出すための構文が **分割代入（Destructuring Assignment）** である。

## 1. オブジェクトの分割代入

オブジェクトから特定のプロパティを取り出す際、従来は `オブジェクト名.プロパティ名` と何度も記述する必要があったが、分割代入を使うことでコードを非常に短くできる。

```javascript
const userProfile = {
  name: "テストユーザー",
  age: 20
};

// 【ドットアクセスを用いた書き方】
const message1 = `名前は${userProfile.name}です。年齢は${userProfile.age}です。`;

// 【分割代入を用いた書き方】（※React開発で推奨）
// オブジェクトのプロパティ名と同じ変数名を {} で囲んで取り出す
const { name, age } = userProfile;
const message2 = `名前は${name}です。年齢は${age}です。`;
```

## 2. 配列の分割代入

配列からのデータ抽出も、インデックス番号（`[0]`, `[1]`）を指定する代わりに、要素の順番通りに変数を割り当てることができる。

```javascript
const userArray = ["テストユーザー", 20];

// 【インデックス番号を用いた書き方】
const message3 = `名前は${userArray[0]}です。年齢は${userArray[1]}です。`;

// 【分割代入を用いた書き方】（※React開発で推奨）
// 配列の順番に合わせて、好きな変数名を [] で定義して取り出す
const [userName, userAge] = userArray;
const message4 = `名前は${userName}です。年齢は${userAge}です。`;
```

## 3. Reactでの活用シーン（Propsの受け取り）
Reactでは、親コンポーネントから渡されるデータ群（Propsと呼ばれるオブジェクト）を受け取る際に、この「オブジェクトの分割代入」がほぼ100%の確率で使用される。
