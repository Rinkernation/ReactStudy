# JS基礎: アロー関数

JavaScriptにおいて、関数をより短く簡潔に記述するための構文が **アロー関数（Arrow Function）** である。
React開発では、コンポーネント自体の定義や、配列の操作（`map` や `filter` など）において非常に頻繁に使用される、現在の標準的な書き方となっている。

## 1. 基本的な構文

従来の `function` キーワードを使わず、`=>`（矢印：アロー）を用いて関数を定義する。
変数（`const`）に関数を代入する形で定義するのが一般的である。

### 従来関数とアロー関数の比較

```javascript
// 【従来の書き方】 functionキーワード
const func1 = function(str) {
  return str;
};
console.log(func1("テスト1"));

// 【推奨される書き方】 アロー関数
const func2 = (str) => {
  return str;
};
console.log(func2("テスト2"));
```

## 2. 記述の省略ルール（アロー関数の最大の利点）

アロー関数は、特定の条件下において、記述を大幅に省略することができる。これによりコードの可読性が格段に上がる。

### ① 処理が1行（単一の式）の場合の省略（暗黙的return）
関数内の処理が「値を返すだけ（1行の式）」の場合、波括弧 `{}` と `return` キーワードを省略し、**矢印の右側に直接戻り値を記述**することができる。

```javascript
// 【省略前】
const add1 = (num1, num2) => {
  return num1 + num2;
};

// 【省略後】波括弧とreturnを省略してスッキリと記述
const add2 = (num1, num2) => num1 + num2;

console.log(add2(10, 20)); // 30
```

### ② オブジェクトを1行で返す場合の省略（要注意）
JavaScriptのオブジェクトは `{}` で定義するため、省略記法を使おうとすると「関数の処理ブロックを表す `{}`」と区別がつかずエラーになってしまう。
そのため、アロー関数でオブジェクトを暗黙的に return させたい場合は、**オブジェクト全体を丸括弧 `()` で囲む**必要がある。

```javascript
// オブジェクトを返す場合は () で囲む
const createProfile = (name, age) => ({
  userName: name,
  userAge: age
});

console.log(createProfile("テストユーザー", 20)); 
// { userName: "テストユーザー", userAge: 20 }
```

## 3. Reactにおける活用例
Reactでは、画面の部品である「コンポーネント」自体をアロー関数で定義するのが主流である。

```javascript
// Reactコンポーネントをアロー関数で定義する例
const App = () => {
  return <div>こんにちは！</div>;
};
```
