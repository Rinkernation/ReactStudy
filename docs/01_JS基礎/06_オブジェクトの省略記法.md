# JS基礎: オブジェクトの省略記法

JavaScriptにおいて、オブジェクトのプロパティ名と、そこに割り当てる変数名が同じである場合に記述を簡略化できる構文が **オブジェクトの省略記法（Shorthand Property Names）** である。
React開発では、コンポーネントに渡すオブジェクトを組み立てる際などに非常によく使われる。

## 1. 基本的な使い方

変数名とプロパティ名が一致している場合、`プロパティ名: 変数名` という記述を、単に `変数名` だけで済ませることができる。

```javascript
const name = "テストユーザー";
const age = 20;

// 【通常の書き方】
const myProfile1 = {
  name: name,
  age: age
};

// 【推奨される書き方】 省略記法
// 名前が同じなら一つにまとめられる
const myProfile2 = {
  name,
  age
};

console.log(myProfile2); // { name: "テストユーザー", age: 20 }
```

## 2. Reactでの活用シーン

Reactでは、複数のデータをまとめて一つのオブジェクトとして子コンポーネントに渡したり、Stateを更新したりする際にこの省略記法が多用される。

```javascript
const [name, setName] = useState("テストユーザー");
const [age, setAge] = useState(20);

// プロパティ名をわざわざ書かなくて良いため、コードがスッキリする
const userInfo = { name, age };
```

このように、名前が重複する無駄な記述を減らすことで、コードの可読性が高まるメリットがある。
