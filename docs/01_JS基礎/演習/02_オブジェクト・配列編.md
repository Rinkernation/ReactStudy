# 演習02: オブジェクト・配列編（分割代入, デフォルト値, 省略記法, スプレッド構文）

オブジェクトや配列の操作、およびモダンな記述方法についての復習問題である。
`vanilla_js/src/practice.js` の中身を一度クリアしてから進めるか、前回のコードをコメントアウトして開始せよ。

---

## 問題4: 分割代入とデフォルト値（オブジェクト）

以下のオブジェクト `user` から、分割代入を使って `id` と `name` を取り出し、`console.log` で出力せよ。
また、`address` が存在しない場合に備えて、デフォルト値として "未設定" を設定し、それも出力せよ。

```javascript
const user = {
  id: 1,
  name: "田中"
};
```

**【期待される出力】**
`1`
`田中`
`未設定`

## 問題5: 分割代入（配列）

以下の配列から、分割代入を使って 1番目の要素を `red`、2番目の要素を `blue` という変数名で取り出し、出力せよ。

```javascript
const colors = ["赤", "青", "黄"];
```

**【期待される出力】**
`赤`
`青`

## 問題6: オブジェクトの省略記法

以下の変数を使用して、プロパティ名と変数名が同じになるようなオブジェクト `myStock` を作成し、出力せよ。
※必ず省略記法を使用すること。

```javascript
const item = "リンゴ";
const price = 150;
```

**【期待される出力】**
`{ item: "リンゴ", price: 150 }`

## 問題7: スプレッド構文（コピーと結合）

以下の指示に従ってコードを書け。

1. 配列 `originalArr` をスプレッド構文を使ってコピーし、新しい配列 `copyArr` を作成せよ。
2. `copyArr` の最初の要素を `100` に書き換えよ。
3. `originalArr` と `copyArr` をそれぞれ出力し、`originalArr` が書き換わっていないことを確認せよ。

```javascript
const originalArr = [1, 2, 3];
```

**【期待される出力】**
`[1, 2, 3]` （元の配列）
`[100, 2, 3]` （コピーした配列）
