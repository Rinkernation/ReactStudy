# 演習03: 実践処理編（map, filter, 三項演算子, 論理演算子）

JS基礎の総仕上げである。実戦で多用する配列処理と条件分岐の復習問題である。

---

## 問題8: mapメソッド

以下の配列 `nameArr` を使用して、それぞれの名前に「さん」を付けた新しい配列 `newNameArr` を作成し、出力せよ。

```javascript
const nameArr = ["田中", "山田", "佐藤"];
```

**【期待される出力】**
`["田中さん", "山田さん", "佐藤さん"]`

## 問題9: filterメソッド

以下の配列 `numArr` から、数値が `5` より大きいものだけを抽出した新しい配列 `newNumArr` を作成し、出力せよ。

```javascript
const numArr = [1, 4, 6, 8, 2, 10];
```

**【期待される出力】**
`[6, 8, 10]`

## 問題10: 三項演算子

以下の変数 `amount` の値が `1000` 以上であれば "予算オーバー"、そうでなければ "予算内" という文字列を `console.log` で出力せよ。
※必ず三項演算子を使用すること。

```javascript
const amount = 1200;
```

**【期待される出力】**
`予算オーバー`

## 問題11: 論理演算子の本当の意味（&& / ||）

以下の指示に従ってコードを書け。

1. 変数 `input` が `null` の場合に、デフォルト値として "未入力です" を代入して出力せよ（`||` を使用）。
2. 変数 `isCheck` が `true` の場合にのみ、"チェック済みです" という文字列を出力せよ（`&&` を使用）。

```javascript
const input = null;
const isCheck = true;
```

**【期待される出力】**
`未入力です`
`チェック済みです`
