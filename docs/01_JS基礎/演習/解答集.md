### 問題4: 分割代入とデフォルト値（オブジェクト）
```javascript
const user = {
  id: 1,
  name: "田中"
};

const { id, name, address = "未設定" } = user;
console.log(id);
console.log(name);
console.log(address);
```

### 問題5: 分割代入（配列）
```javascript
const colors = ["赤", "青", "黄"];

console.log(red);
console.log(blue);
```

### 問題6: オブジェクトの省略記法

## 03_実践処理編

<summary>解答を表示する</summary>

### 問題8: mapメソッド
```javascript
const nameArr = ["田中", "山田", "佐藤"];
const newNameArr = nameArr.map((name) => `${name}さん`);
console.log(newNameArr);
```

### 問題9: filterメソッド
```javascript
const numArr = [1, 4, 6, 8, 2, 10];
const newNumArr = numArr.filter((num) => num > 5);
console.log(newNumArr);
```

### 問題10: 三項演算子
```javascript
const amount = 1200;
const message = amount >= 1000 ? "予算オーバー" : "予算内";
console.log(message);
```

### 問題11: 論理演算子の本当の意味
```javascript
const input = null;
const fee = input || "未入力です";
console.log(fee);

const isCheck = true;
const message = isCheck && "チェック済みです";
console.log(message);
```
