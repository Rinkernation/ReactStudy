# 演習01: 基礎構文編（変数, テンプレート文字列, アロー関数）

これまでに学んだ「変数宣言」「テンプレート文字列」「アロー関数」の復習問題である。
`vanilla_js/src/practice.js` にコードを書き、`console.log` で期待通りの結果が表示されるか確認せよ。

---

## 問題1: 変数宣言（const / let）

以下の指示に従ってコードを書け。

1. 再代入が可能な変数 `count` を数値の `1` で定義せよ。
2. `count` に `2` を再代入せよ。
3. 再代入が不可能な変数 `userName` を文字列の "テストユーザー" で定義せよ。
4. それぞれの値を `console.log` で出力せよ。

## 問題2: テンプレート文字列

以下の変数を使用して、期待される出力結果を `console.log` で表示せよ。
※必ずテンプレート文字列（バッククォート）を使用すること。

```javascript
const name = "佐藤";
const age = 25;
```

**【期待される出力】**
`私の名前は佐藤です。年齢は25歳です。`

## 問題3: アロー関数

以下の従来の関数を、アロー関数に書き換えよ。
また、余計な記述（波括弧やreturn）を省ける場合は省略記法を使用せよ。

```javascript
// この関数をアロー関数に書き換える
function multiply(num) {
  return num * 2;
}
```

**【確認用コード】**
`console.log(multiply(10)); // 20 と表示されれば正解`
